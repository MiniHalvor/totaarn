Husk: Bilder og statiske assets legges i static foldern.


GITHUB
git status -> sier seg selv
1. git add . -> gjør klar alle redigerte filer for push
2. git commit -m "melding" -> committer til branchen med melding
3. git push -u origin main -> pusher endringer til main branchen, ferdig
4. git status -> dobbeltsjekk at alt gikk greit


NPM:

npm run dev ->  vet ikke
npm run build -> kjør buildet
npm run preview -> Få localhost opp


Finder:

cmd + shift + . -> vis skjulte filer, greit hvis du har fucka opp git


MacOS:

Fn + f -> toggle fullscreen for window



Chrome:

cmd + option + j -> javascript konsoll i chrome. Skriv document.title = "navn" for å endre navn på tabbem. Veldig greit hvertfall for chatgpt
cmd+t -> åpne ny tab
control + tab -> gå til fanen til høyre
control + shift + tab -> Gå til fanen til venstre
cmd + *1-9* ->  for å gå til en spesifikk fane
cmd + w -> lukk tab
